Vue d'ensemble

Parcours Moniteur est un back-office cartographique reserve aux administrateurs. Il permet de consulter les points d'interet pedagogiques, de filtrer les POI en cours de validation, d'ouvrir leur detail depuis la carte ou la liste, et de modifier leurs informations.

Acces Connexion reservee aux administrateurs autorises.
Usage Consulter, filtrer et corriger les points d'interet.
Carte Voir les POI sur la carte et ouvrir leur fiche en un clic.

Acces administrateur

  1. Ouvrez l'application Parcours Moniteur.
  2. Saisissez votre email administrateur.
  3. Saisissez votre mot de passe.
  4. Cliquez sur Se connecter.
  5. Si votre compte est autorise, le back-office s'ouvre automatiquement.
Si la connexion est refusee, contactez la personne responsable du compte administrateur. Le back-office n'est pas accessible avec un compte standard.

Carte et liste des POI

Liste

  • La liste affiche les POI disponibles dans le back-office.
  • Chaque item affiche le nom, le type et un badge En cours si le statut correspond.
  • Les coordonnees ne sont plus affichees dans la liste.
  • Un clic sur un item ouvre le panneau de detail du POI.

Carte

  • La carte affiche les POI positionnes geographiquement.
  • Les pins cyan representent les POI actifs.
  • Les pins jaunes representent les POI avec le statut en cours.
  • Les pins rouges representent les POI inactifs.
  • Un clic sur un pin ouvre le detail dans le panneau lateral, sans modal au-dessus du pin.

Filtre En cours

  • Le bouton En cours controle la liste et la carte en meme temps.
  • Filtre desactive : tous les POI sont visibles dans la liste et sur la carte.
  • Filtre actif : seuls les POI en cours sont visibles.

Detail d'un POI

Le detail s'ouvre depuis un clic sur la liste ou sur un pin de la carte.

  • Le titre affiche le nom du POI.
  • Le sous-titre affiche le statut du POI.
  • La section Type affiche le type du POI.
  • La section Description affiche la description ou un message par defaut.
  • La position exacte n'est pas affichee sous forme de coordonnees.
  • Le bouton Modifier ouvre le formulaire d'edition.
  • Le bouton Retour revient a la liste et deselectionne le POI.

Modification d'un POI

Le formulaire permet de modifier les informations fonctionnelles du POI.

  • Nom est obligatoire.
  • Type est obligatoire.
  • Statut est obligatoire.
  • Description est optionnelle.
  • Les coordonnees ne sont pas affichees dans le formulaire.

Deplacement sur la carte

  1. Ouvrir le detail d'un POI.
  2. Cliquer sur Modifier.
  3. Cliquer sur la carte a la nouvelle position souhaitee.
  4. Cliquer sur Enregistrer.

Meme si les coordonnees ne sont pas visibles, l'application conserve le comportement : le clic sur la carte met a jour la position sauvegardee du POI.

Suppression d'un POI

Le bouton Supprimer demande une confirmation navigateur avant suppression definitive du POI. Utilisez cette action uniquement si le point ne doit plus etre conserve.

Statuts et couleurs

  • Actif : POI visible avec un pin cyan.
  • En cours : POI a verifier ou en cours de traitement, visible avec un pin jaune.
  • Inactif : POI conserve mais non actif, visible avec un pin rouge.
Le statut peut etre modifie depuis le formulaire de modification d'un POI.

Aide rapide

Je ne peux pas me connecter

  • Verifiez votre email et votre mot de passe.
  • Verifiez que votre compte a bien un acces administrateur.
  • Si le probleme persiste, contactez le responsable de l'application.

Je ne vois pas tous les POI

  • Verifiez si le filtre En cours est actif.
  • Desactivez le filtre pour afficher tous les POI.
  • Utilisez le bouton d'actualisation pour recharger les donnees.

Un POI n'apparait pas sur la carte

  • Verifiez si le filtre En cours masque ce POI.
  • Le POI peut ne pas avoir encore de position definie.
  • Ouvrez sa fiche, passez en modification, puis cliquez sur la carte pour placer le point.
