Acces administrateur

Le back-office est reserve aux comptes presents dans la table users avec is_admin = true et is_active = true.

Points d'interet

  1. Utilisez le bouton d'actualisation pour recharger les POI depuis Supabase.
  2. Cliquez sur un POI dans la liste ou sur un pin de la carte pour ouvrir son detail.
  3. Depuis le detail, utilisez Modifier pour changer les informations du POI.
  4. En mode modification, cliquez sur la carte pour deplacer le point.

Statuts

  • en cours indique un POI en attente ou a verifier.
  • Les POI en cours sont marques avec l'accent jaune.
  • Les POI actifs ou valides utilisent le bleu principal.
